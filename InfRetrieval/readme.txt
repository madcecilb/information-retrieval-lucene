command to run program:
	java -jar <Assignment-1.jar location> <reut2-000.xml location> <query>
for example: 
	java -jar C:\Assignment-1.jar C:\reut2-000.xml information
by default program searches for a query: 
	date:[19870301 TO 19870302] AND information retrieval
so, command:
	java -jar C:\Assignment-1.jar C:\reut2-000.xml
searches for:
	date:[19870301 TO 19870302] AND information retrieval
	
file locations:
	Assignment-1.jar inside folder /runnable_jar
	reut2-000.xml inside folder /file
	documentation inside folder /doc
